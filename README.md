# ❤️ Heart Disease Prediction

## 📌 Overview

This project predicts the likelihood of heart disease using machine learning techniques.

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost

## 🧠 Models Implemented

* Logistic Regression
* Random Forest
* XGBoost

## 🔧 Steps Performed

* Data Cleaning (handled missing values using mean and mode)
* Feature Engineering (converted categorical data using one-hot encoding)
* Model Training using multiple algorithms
* Model Evaluation using accuracy and cross-validation

## 📊 Results

* Logistic Regression Accuracy: ~82%
* Random Forest Accuracy: ~85%
* XGBoost Accuracy: ~85%
* Cross Validation Accuracy: ~82.5%

## 📌 Key Insight

The model was improved by reducing overfitting using stratified sampling and hyperparameter tuning, resulting in stable and reliable performance.

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Open the notebook:
   jupyter notebook heart_disease.ipynb
